# Nexo

Nexo es un asistente ligero para Windows que permite consultar, entender y controlar el equipo sin abandonar lo que estás haciendo.

## Estado

El proyecto se encuentra en desarrollo temprano.

## Objetivos iniciales

- Abrir Nexo con Alt + A.
- Usar IA mediante texto o voz.
- Ejecutar comandos locales sin gastar tokens.
- Controlar el audio por aplicación.
- Crear atajos personalizados.
- Mostrar información útil del sistema.
- Mantener un consumo bajo.

## Tecnología

- C#
- WPF
- .NET 10
- Windows 11
