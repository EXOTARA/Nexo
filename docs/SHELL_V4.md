# Nexo Shell v4

Este corte transforma el panel lateral anterior en un workspace real.

## Principios

- `Ctrl + Espacio` sigue siendo una paleta independiente.
- El workspace se abre solo para contenido que necesita profundidad.
- La navegación expandida desplaza el contenido; nunca lo cubre.
- Inicio resume pendientes, enfoque, contexto y actividad reciente.
- Las acciones locales siguen evitando el LLM.
- El movimiento usa posición y opacidad, no escala.

## Estados actuales

1. Cápsula para confirmaciones breves.
2. Paleta para comandos y consultas.
3. Workspace para módulos, conversación y configuración.

## Próximo bloque

Resource Governor y Modo Juego: carga de GPU, pantalla completa, batería y degradación segura.
