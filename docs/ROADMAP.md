# Roadmap de Nexo

Este documento muestra hacia dónde va el proyecto. Las prioridades pueden cambiar conforme probemos la aplicación.

## Completado

### Base del proyecto

- Solución modular con Nexo.App, Nexo.Core y Nexo.Windows.
- Pruebas automatizadas.
- Repositorio y flujo mediante Pull Requests.

### Interfaz

- Barra lateral sin bordes.
- Animaciones ligeras.
- Navegación modular.
- Temas y contraste global.
- Personalización persistente.
- Modo Peek.

### Sistema

- CPU.
- RAM.
- GPU.
- VRAM.
- Almacenamiento.
- Proceso con mayor uso de memoria.

### Comandos

- Intérprete inicial de lenguaje natural.
- Comandos locales para abrir módulos y terminales.
- Cápsulas de confirmación.
- Historial opcional de conversación.

## En desarrollo

### Sprint 3B — Audio

- Volumen general.
- Sesiones de audio por aplicación.
- Barras de volumen individuales.
- Silenciar aplicaciones.
- Ejecutar comandos como:
  - `baja Spotify`;
  - `pon Discord al 25 %`;
  - `silencia Zen`;
  - `baja todo menos Discord`.

## Próximamente

### Voz

- Pulsar para hablar.
- Reconocimiento de voz en español.
- Respuestas habladas.
- Palabra de activación opcional `Nexo`.

### IA

- Sistema de proveedores.
- OpenAI.
- Modelos locales mediante Ollama o LM Studio.
- Contexto autorizado del sistema.
- Acciones sugeridas con confirmación.

### Capturas

- Captura por región.
- Texto, flechas y formas.
- Pixelado.
- OCR.
- Análisis mediante IA.
