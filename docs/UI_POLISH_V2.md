# Nexo UI Polish v2

## Principios

- Nexo aparece rápido y desaparece sin robar atención.
- El movimiento comunica continuidad; no decora ni permanece activo sin razón.
- La paleta es la superficie primaria. El panel completo solo se usa cuando la tarea necesita profundidad.
- Los comandos conocidos se ejecutan localmente. La IA no explica cómo abrir algo que Nexo puede abrir.
- Los controles abstractos pueden usar el glifo floral de Nexo; las acciones concretas conservan nombres o símbolos claros.
- Contraste legible en tooltips, listas, menús y selectores, sin superficies blancas heredadas de WPF.
- La personalización avanzada no debe complicar los primeros minutos de uso.

## Interacción de la paleta

- `Ctrl + Espacio`: abrir.
- Escribir `E`: completa **Explorador de archivos**.
- `D`, `V`, `C`, `T`, `P`, `M` y `F`: accesos rápidos equivalentes.
- `↑` / `↓`: cambiar selección.
- `Tab`: completar la sugerencia seleccionada.
- `Enter`: ejecutar una coincidencia clara o enviar la consulta exacta.
- `Ctrl + Enter`: forzar el envío como consulta.
- `Shift + Enter`: insertar una nueva línea.
- `Alt + Enter`: abrir el espacio completo de Nexo.
- `Esc`: cerrar.

## Movimiento

- Una sola transición dominante por interacción.
- Desplazamientos cortos y opacidad sutil.
- Sin animaciones infinitas.
- El preset **Sin movimiento** conserva toda la funcionalidad.
